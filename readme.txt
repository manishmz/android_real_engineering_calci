# Just import thisproject in your ADT bundle for android
and run in your emulator
Description
It is an Calci app useful for engineers.
It contains probably lots of functions as in Real calci.
 Includes 
sin(),cos(),tan(),asin(),acos(),atan(),p
degree,radian mode,exponential,x!,log(),ln(),%
Containing Functions such as
  Integration
  Matrix Operations
     add and sub
     inverse
     multiplication
     determinant
  Exp in x
  nPr
  nCr
Extra Features:
   Unit Conversion 
   History
   Text to Speech
